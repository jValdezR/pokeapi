# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar
```
npm install
```
3. Instalar Nest CLI
```
npm i -g @nest/cli
```
4. Levantar la db
```
docker-compose up -d
```
5. Reconstruir la db
```
localhost:3000/seed/
```

## Stack usado
* MongoDB
* Nest# pokeapi
